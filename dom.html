Algunos métodos para interactuar con el DOM:

  • querySelector() devuelve el código HTML del primer elemento que tenga el valor del selector CSS buscado.
  • querySelectorAll() devuelve en un arreglo el código HTML de todos los elementos que tengan el valor del selector buscado.
  • getElementById() devuelve el código HTML del elemento con el nombre de id buscado, este método es más rápido que querySelector.
  • innerText Crea o modifica el contenido textual del elemento especificado.
  • innerHTML Crea o modifica el contenido HTML del elemento especificado.
  • createElement() Crea un nuevo elemento HTML.
  • appendChild() Agrega un nodo hijo al final del elemento HTML especificado.
  • removeChild() Elimina el nodo hijo indicado del elemento HTML especificado.
  • insertBefore() Inserta un nuevo nodo hijo antes del indicado como referencia en elemento HTML especificado.
  • insertAdjacentHMTL() Inserta código HTML dada una posición relativa a un elemento y éstas pueden ser:
    • beforebegin: Antes que el elemento mismo.
    • afterbegin: Justo al inicio dentro del elemento, antes del primer elemento hijo.
    • beforeend: Dentro del elemento, justo después del último elemento hijo.
    • afterend: Después del elemento mismo.
